# 2026-04-26 微云备份脚本

## 任务
为手提电脑恢复 OpenClaw Agent 文件准备备份脚本，并设置实时监控备份方案。

## 完成内容

### 1. 恢复脚本 (restore_from_weiyun.ps1)
上传到微云 OpenClaw_Agent_Backup 文件夹。

功能：
- 自动检查 mcporter 是否安装
- 自动查找微云备份文件夹
- 下载所有备份文件到正确位置
- 处理 .openclaw/ 前缀路径映射

使用方式（手提电脑）：
1. 安装 QClaw
2. 从微云下载 restore_from_weiyun.ps1
3. 用 PowerShell 运行

### 2. 实时监控脚本 (watch_and_backup.ps1)
上传到微云 OpenClaw_Agent_Backup 文件夹。

功能：
- FileSystemWatcher 监听工作区文件变化（.md/.json/.yml/.yaml/.txt）
- 5秒防抖：连续改动只上传一次
- 后台异步上传，不阻塞监控
- 按 Ctrl+C 停止

使用方式（手提电脑）：
1. 确认 mcporter 已配置（mcporter auth weiyun）
2. 从微云下载 watch_and_backup.ps1
3. 用 PowerShell 运行（会持续后台运行）

注意：脚本中硬编码了上传脚本路径和 token，上传到微云后需要在恢复的目标电脑上：
- 确认 upload_to_weiyun.py 在 `D:\QClaw\resources\openclaw\config\skills\weiyun\scripts\upload_to_weiyun.py`
- 或修改脚本中的 $uploadScript 变量指向正确位置
